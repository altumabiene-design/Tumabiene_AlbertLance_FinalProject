# Expense Tracker (CLI)
## Description
The Expense Tracker is a command-line Python application that allows users to record, manage, and analyze their daily expenses. 

Each expense includes a unique ID, title, amount, category, and date, enabling better tracking and organization of financial data. The system also stores data in a JSON file to ensure persistence.


# Features
- Add expense with ID, title, amount, category, and date
- View all recorded expenses
- Search expenses by keyword
- Delete expenses by title or ID
- Sort expenses by amount
- Calculate total expenses
- Generate category-based summary
- Save and load data using JSON file
